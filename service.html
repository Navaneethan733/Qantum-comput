/ System Flow /

How a circuit runs.

From local Python scripts to cooled hardware measurements, trace the path of your computation.

Design & Submit

Submit circuits remotely via the Stackly API using standard Qiskit or Cirq, or upload compiled assembly code.

Calibrate & Route

Our system schedules the job and routes it using the transpiler, mapping layout elements onto optimized physical channels.

Hardware Execution

Control lines deliver microwave pulses to superconducting Josephson junction qubits in dilution refrigerators cooled to 15mK.

Measure & Return

Readout resonators capture state measurements. Results and live system fidelity telemetry are returned to your console.
